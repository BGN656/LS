Робота з gihub

Статусы файла

Untracked -  добавили файл
Staged - после  git add
Modified - после изменения существующего файла
Commited -  после комита
 

Отмена изменения

Git clean -f //Untracked

Git restore  <файл> //Modified 
Git restore . //Modified 

Git restore --staged //Staged 
Git reset   //Staged 


Git reset HEAD~1 //Commited
